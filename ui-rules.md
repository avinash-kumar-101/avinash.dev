# UI Rules

## Layout

- Maintain consistent section spacing.
- Avoid unnecessary dividers and partitions.
- Content and images should blend naturally.

---

## Cards

- Use the same padding.
- Maintain consistent border radius.
- Use similar hover effects.

---

## Buttons

- Maintain the same height and border radius.
- Keep transitions smooth.

---

## Images

- Images should feel integrated with the content.
- Avoid awkward placement.
- Maintain consistent image corner radius.

---

## Icons

Use only one icon library throughout the project.

Preferred:

- Lucide React

---

## Hover Effects

- Use subtle scaling.
- Avoid exaggerated animations.

---

## Animations

Preferred Duration:

300ms

Preferred Easing:

ease-in-out

---

## Responsive Behaviour

Desktop First.

### Tablet

Maintain layout consistency.

### Mobile

Stack elements naturally without breaking design hierarchy.

---

## Important

Never redesign existing sections unnecessarily.

Maintain visual consistency across every page.
