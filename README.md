## 📌 Overview

A **production-grade personal portfolio** built with modern frontend tooling — designed to be fast, animated, and fully responsive across all devices. This isn't just a static CV page; it's a fully engineered React SPA with structured sections, optimized assets, smooth Framer Motion animations, and a live working contact form.

Built to reflect real-world frontend development practices — clean component architecture, reusable UI patterns, and performance-first design decisions.

---

## ✨ Features

- ⚡ **Blazing fast** — Built with Vite 7 + React 19 for optimal HMR & production build performance
- 🎨 **Fully animated** — Page-level & element-level animations using Framer Motion
- 📱 **Responsive** — Mobile-first layout that works seamlessly on all screen sizes
- 📬 **Functional contact form** — Integrated with EmailJS (no backend required)
- 🗂️ **Modular architecture** — Each section is an isolated component for easy maintenance
- 🏆 **Certificates section** — Showcases verified certifications with official issuer details
- 🛠️ **Skills visualization** — Categorized tech stack with proficiency indicators
- 💼 **Experience timeline** — Internship & project highlights in a structured layout
- 🔒 **Zero backend dependency** — Fully static, deployable on any CDN/edge network

## 🚀 Getting Started

### Prerequisites
- Node.js `>= 18.x`
- npm `>= 9.x`

### Clone & Install

```bash
# Clone the repository
git clone https://github.com/avinash-kumar-101/avinash.dev.git

# Navigate to project directory
cd avinash.dev

# Install dependencies
npm install
```

### Development Server

```bash
npm run dev
# → http://localhost:5173
```

### Production Build

```bash
npm run build       # Outputs to /dist
npm run preview     # Preview production build locally
```

### Lint

```bash
npm run lint
```

## 🌐 Deployment

This project is deployed on **[Vercel](https://vercel.com)** with automatic CI/CD triggered on every push to the `main` branch.

| Branch | Environment | URL |
|--------|-------------|-----|
| `main` | Production | [avinashdev.vercel.app](https://avinashdev.vercel.app/) |

To deploy your own fork:

```bash
npm install -g vercel
vercel --prod
```
