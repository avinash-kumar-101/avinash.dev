# Animation Guidelines

## Goal

Use subtle and smooth animations that enhance the user experience.

Animations should never distract from the content.

Consistency is more important than creativity.

---

# General Principles

- Use animation only when it improves usability.
- Avoid flashy or excessive motion.
- Keep all animations smooth and professional.
- Similar components should have similar animations.

---

# Duration

Fast:
200ms

Normal:
300ms

Slow:
500ms

Avoid animations longer than 700ms.

---

# Easing

Preferred easing:

ease-out

For spring animations:

stiffness: 100
damping: 20

Maintain the same easing throughout the website.

---

# Scroll Reveal

Use subtle fade-in and translateY animations.

Initial:

opacity: 0
y: 20

Animate:

opacity: 1
y: 0

Duration:

0.5s

Trigger only once.

Avoid excessive scroll animations.

---

# Hover Effects

Cards:

scale: 1.02

Buttons:

scale: 1.03

Icons:

rotate slightly or scale subtly.

Avoid large scaling effects.

---

# Hero Section

- Use slightly slower animations.
- Content should appear naturally.
- Avoid dramatic entrances.

---

# Sections

Animate sections sequentially.

Recommended order:

1. Badge
2. Heading
3. Description
4. Buttons
5. Images

Maintain consistent delay between elements.

---

# Cards

Project cards:

fade-up animation

Skill cards:

fade-up animation

Experience cards:

fade-up animation

Use the same animation pattern everywhere.

---

# Images

Images should:

- fade in
- scale slightly from 0.98 to 1

Avoid rotating or bouncing images.

---

# Navigation

Navbar should:

- remain smooth
- have subtle background transition

Avoid complicated navbar animations.

---

# Page Transitions

Keep transitions fast and minimal.

Avoid excessive page loaders.

---

# Important Rules

Never use:

- Bounce animations
- Flashing effects
- Infinite animations
- Large rotations
- Dramatic scaling
- Different animation styles for similar components

Maintain one animation language across the entire portfolio.

Animations should feel premium, modern, and natural.
