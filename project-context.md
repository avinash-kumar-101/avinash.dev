# Portfolio Context

## Purpose

Build a premium and modern personal portfolio website.

---

## Overall Feel

The website should feel:

- Modern
- Premium
- Clean
- Professional
- Minimal

---

## Design Philosophy

Consistency is more important than creativity.

Every section should feel like part of the same product.

---

## Style

Avoid:

- Random colors
- Inconsistent typography
- Uneven spacing
- Excessive shadows
- Over-the-top animations

---

## Components

All sections should share the same design language.

Including:

- Hero Section
- About Section
- Skills Section
- Certificates Section
- Experience Section
- Projects Section
- Contact Section
- Footer

---

## Animation Philosophy

Animations should enhance the experience, not distract from it.

Use subtle motion only.

---

## Images

Images should complement the content and blend naturally into the layout.

Avoid creating a partitioned appearance between text and images.

---

## Priority

1. Consistency
2. Readability
3. User Experience
4. Visual Hierarchy
5. Creativity
